Albatour

Project Brief

This project is a project from the UI/UX competition organized by @technofest.sk at university level in 2022 with the theme " Application providing transportation services for travelers and friendly for students nomads ".

Albatour is a traveling app that provides information services regarding ordering vehicle tickets, rooms, hotels or rentals electronically. Vision to be a traveling application that can provide the best easy experience, and can be used by all groups. Mission become the best application by always presenting new things with unique features.

My Role

UI Design, Logo Design

Tools

Figma, Miro, Whimsical, Notion

Timeline

2 Weeks

My task as a UI Designer in this project before designing the interface is of course to create several elements wrapped in a design system so that this UI is more consistent. The design system elements such as, Grid, Colors, Typography, Iconography, Spacing, Illustrations, and Buttons.

What's in a logo?

This logo is inspired by a bird called the Albatross. The albatross is considered an explorer bird due to its incredible migratory habits with the widest wingspan in the world.

The choice of lavender indigo purple reflects the spirit of adventure and courage in exploring new and unknown places.

Wireframe

Explore Concept

Creating wireframes in this project is not my main task, however, creating wireframes is a common task, as it helps in planning and organizing UI elements.

In engaging in wireframe creation, as a UI Design role I have to pay attention to the visual aspect, while UX Design pays attention to the functional and interaction aspects, so that there is a balance between the two aspects.
